# ABC Challenge

ABC Challenge es una página web interactiva creada para que niños y niñas puedan aprender las letras del abecedario de una forma visual, sencilla y entretenida.

Este proyecto fue desarrollado en equipo como parte de un laboratorio de HTML, CSS, JavaScript y GitHub. Cada integrante trabajó en un grupo de letras diferente y, al finalizar, reunimos todos los aportes en una sola página.

## ¿Cómo funciona?

La página contiene las 26 letras del abecedario.

Cada letra aparece dentro de una card. Al hacer clic sobre ella, la card se voltea y muestra una imagen junto con una palabra relacionada.

Por ejemplo:

- F de Flor
- G de Gato
- H de Hormiga
- I de Iguana
- J de Jirafa

La página también cuenta con un contador que registra cuántas letras diferentes ha descubierto el usuario.

## Funcionalidades

- Barra de navegación con el nombre del proyecto.
- Contador de letras descubiertas.
- Una card para cada letra del abecedario.
- Efecto de volteo al hacer clic.
- Imágenes y palabras relacionadas con cada letra.
- Filtro para mostrar solamente las vocales.
- Botón para volver a mostrar todas las letras.
- Sección con los nombres de los integrantes del equipo.
- Diseño pensado para ser claro, llamativo y fácil de utilizar.
